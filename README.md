# Counter-App
A web app made using typescript,html and scss

## Description
A counter app 

## Requirements
* [Node Js v14+](https://nodejs.org/en/download)

## Installation instructions
### Install TypeScript
```bash
npm install -g typescript

```

### Install SCSS
```bash
npm install node-sass
```

## Running on local

Compile app.ts
```bash
tsc app.ts

```


Compile app.ts in watch mode (optional)
```bash
tsc app.ts -w

```
